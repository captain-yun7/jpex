프로젝트명 입시 정보 시스템 백엔드 API 개발
예상 기간 4주
유효기간 견적일로부터 30일

프로젝트 개요

프로젝트 목적
  • 클라이언트 사이드 데이터를 서버 기반 시스템으로 전환
  • 사용자 인증 및 개인화된 입시 정보 제공
  • 대학 입시 정보 데이터베이스 구축
  • 합격 가능성 분석 및 맞춤 추천 시스템
기술 스택
  • Runtime: Node.js v18+
  • Framework: Express.js 4.x
  • Database: PostgreSQL 15+ / MySQL 8+
  • ORM: Prisma 5.x / Sequelize 6.x

개발 범위

Phase 1: 기본 인프라
  • 프로젝트 기본 구조 설정
  • 데이터베이스 구축 및 ORM 설정
  • 마스터 데이터 임포트 (대학/학과 정보)
  • 사용자 인증 시스템 (회원가입/로그인/JWT)
  • 인증 미들웨어 및 권한 관리
Phase 2: 핵심 기능
  • 성적 관리 API (CRUD, 환산점수 계산)
  • 대학/학과 검색 API (통합 검색, 필터링)
  • 페이지네이션 및 정렬 기능
  • 합격 가능성 분석 API (안정/적정/소신 판정)
  • 맞춤 대학 추천 알고리즘
  • 점수 분포 데이터 분석
Phase 3: 고급 기능
  • 포트폴리오 관리 API (생성/수정/삭제)
  • 포트폴리오 자동 생성 기능
  • 포트폴리오 공유 기능
  • 대학/학과 비교 API
  • 알림 시스템 API (상담 내역, 입시 정보 알림)
  • 사용자 선호도 관리
프론트엔드 연동
  • API 클라이언트 구현 (Axios)
  • 기존 LocalStorage 코드를 API 연동으로 전환
  • 에러 처리 및 로딩 상태 관리
  • 인터셉터 구현 (인증 토큰 자동 추가)
테스트 & QA
  • 단위 테스트
  • E2E 테스트
배포 & 운영
  • 클라우드 인프라 구축 및 배포
