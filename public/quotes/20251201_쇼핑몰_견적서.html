프로젝트 개요

Spring Boot 기반의 쇼핑몰 웹사이트 개발 및 AWS 클라우드 인프라 구축

예상 기간

약 4~6주

기술 스택

Frontend React / S3 + CloudFront + Route53
Backend Spring Boot + MyBatis / EC2 (혹은 ECS 및 EKS)
Database Oracle / RDS

개발 범위

  • 회원 관리 - 회원가입, 로그인, 마이페이지
  • 상품 관리 - 상품 목록, 상세페이지, 카테고리
  • 장바구니/주문 - 장바구니, 주문/결제, 주문내역
  • 관리자 - 상품/주문/회원 관리 페이지
  • 인프라 구축 - AWS 환경 세팅 및 배포
