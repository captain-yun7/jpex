프로젝트 개요

사업장 방문객을 위한 인터랙티브 키오스크 시스템 개발

디스플레이 사양

3120 x 624 px 터치스크린 LED

구동 환경

미니PC (Windows) 키오스크 모드

예상 기간

약 8주

화면 구성

  • 대기화면 - 미디어 아트 영상 자동재생, 터치 안내
  • 메인 메뉴 - HISTORY, AFFILIATES, TECH, MEDIA 4개 섹션
  • HISTORY 페이지 - 타임라인 레이아웃 (1996~2025년)
  • AFFILIATES 페이지 - 글로벌 네트워크 지도 및 지점 카드
  • TECH 페이지 - 제품/기술 그리드 (2열, 4~6개 제품)
  • MEDIA 페이지 - 미디어 자료 그리드 (3열, 6~9개 콘텐츠)

주요 기능

  • 터치 이벤트 처리 및 화면 전환 애니메이션
  • 30초 무조작 시 자동 대기화면 복귀
  • 3120 x 624 해상도 최적화
  • 파란색 계열 테크 디자인
  • 미니PC 키오스크 모드 구동 설정
