프로젝트 개요

렌터카 사업 운영을 위한 고객용 렌터카 사이트와 내부 장기렌터카 견적 웹 프로그램 2개 시스템을 구축합니다.

(1) 렌터카 고객 사이트

코웨이몰 렌탈 사이트와 유사한 구조의 렌터카 전문 웹사이트

  • 메인 페이지 (차량 카탈로그, 프로모션 배너, 상담 신청)
  • 차량 목록 및 상세 페이지 (차종별 분류, 옵션, 렌탈료 안내)
  • 렌탈 상담 신청 폼 (이름, 연락처, 관심 차종, 렌탈 기간)
  • 이벤트/혜택 페이지
  • 고객 후기 및 FAQ
  • 회사 소개 및 오시는 길
  • 반응형 웹 디자인 (PC / 모바일)
(2) 장기렌터카 견적 웹 프로그램 (내부용)

기존 엑셀 기반 견적 시스템을 웹으로 전환한 내부 업무 시스템

  • 직원 및 협력업체 계정 발급 (관리자만)
  • 차량 관리 / 등록 (차종, 옵션, 가격 정보)
  • 견적 입력 시 견적서 PDF 자동 생성
  • PDF 출력 / 인쇄 기능
  • 반응형 웹
