프로젝트 개요

에스킴컴퍼니 웹사이트의 검색 노출 성능을 근본적으로 개선합니다. 현재 동적 파라미터 기반 URL을 검색엔진 친화적 구조로 전환하고, 포트폴리오 콘텐츠가 Google·Naver에서 효과적으로 색인될 수 있도록 메타 정보 관리, 콘텐츠 분류 체계, 소셜 공유 최적화까지 일괄 적용합니다.

현재 상태 및 개선 방향
  • URL 구조 개선: portfolio.php?idx=52 같은 동적 URL을 /portfolio/puma-seongsu-campaign 형태의 의미 기반 URL로 전환하여 검색엔진 인식률 향상
  • 콘텐츠 분류 체계 구축: 지역별·업종별 2단계 카테고리 구조를 도입하고, 같은 분류 내 관련 콘텐츠를 자동 추천하여 사이트 체류 시간 증가
  • 검색 결과 품질 관리: 각 콘텐츠별 검색 결과에 노출되는 제목·설명을 직접 관리할 수 있는 기능 추가
  • 소셜 공유 대응: 카카오톡·링크드인 등에서 링크 공유 시 브랜드 이미지에 맞는 미리보기 자동 생성
  • 검색엔진 색인 자동화: 새 콘텐츠 발행 시 검색엔진에 자동 알림하여 빠른 노출 확보
