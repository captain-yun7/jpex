프로젝트 개요

번역 서비스 소개 및 견적 요청이 가능한 기업형 홈페이지 구축

예상 기간

약 3~4주

개발 범위

  • 메인 페이지
    • 회사 소개 및 핵심 서비스 안내
    • 주요 서비스 영역 배너
    • 빠른 견적 요청 CTA
  • 서비스 소개 페이지
    • 번역 서비스 종류별 상세 안내
    • 지원 언어 및 분야 소개
    • 작업 프로세스 안내
  • 견적 요청 시스템
    • 온라인 견적 요청 폼
    • 파일 첨부 기능
    • 이메일 알림 연동
  • 회사 소개
    • 회사 연혁 및 비전
    • 오시는 길 (지도 연동)
  • 관리자 기능
    • 견적 요청 관리
    • 콘텐츠 관리 (서비스, 공지사항 등)
기술 스택

Next.js 15+, TypeScript, Tailwind CSS, PostgreSQL

견적 내역

항목 세부내용 금액(원)
홈페이지 제작 메인, 서비스 소개, 회사 소개 페이지 400,000
견적 요청 시스템 폼, 파일첨부, 이메일 알림 300,000
