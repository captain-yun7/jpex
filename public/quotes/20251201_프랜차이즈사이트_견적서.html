프로젝트 개요

프랜차이즈 브랜드 소개 및 가맹점 안내를 위한 홈페이지 제작

참고: 육회바른연어 (yookbarun.com) 스타일

예상 기간

약 4~5주 (협의 시 단축 가능)

기술 스택

Frontend Next.js + Tailwind CSS / Vercel 배포
Backend Next.js API Routes (서버리스)
Database Supabase 또는 Neon (PostgreSQL)

※ 기능 복잡도가 올라갈 시 별도 백엔드 분리 가능

개발 범위

  • 브랜드 소개 - 브랜드 스토리, 철학, 비전
  • 메뉴 소개 - 메뉴 목록, 상세 이미지
  • 매장 안내 - 매장 검색, 지도 연동
  • 가맹 문의 - 가맹 절차, 문의 폼
  • 공지/뉴스 - 소식, 이벤트 게시판
  • 관리자 - 콘텐츠 관리 페이지

견적 내역

항목 세부내용 금액(원)
기획 및 디자인 UI/UX 설계, 반응형 디자인 500,000
웹 개발 프론트엔드 + 백엔드 + 관리자 1,300,000
배포 및 세팅 Vercel 배포, 도메인 연결, DB 구축 200,000
총 금액 2,000,000

유지보수 (선택)

개발 완료 후 유지보수 방식을 선택하실 수 있습니다.

옵션 1. 건별 방식

요구사항별 비용 산정 (기능 추가, 수정, 페이지 추가 등)

옵션 2. SLA 방식 (월단위 유지보수)

월 10만원~ / 간단한 기능 수정 포함

※ 사후 관리는 계약 형태와 관계없이 무기한 책임지고 운영합니다.

기타 사항

  • 개발 완료 후 1개월간 무상 하자보수를 제공합니다.
  • 콘텐츠(텍스트, 이미지)는 의뢰인이 제공합니다.
  • 추가 기능 개발은 별도 협의가 필요합니다.
  • ※ 서버 호스팅: 연 20만원 (도메인 + 호스팅 + 장애 시 복구)

참고사항

  • 본 견적서는 작성일로부터 30일간 유효합니다.
  • 프로젝트 진행 중 요구사항 변경 시 일정과 비용이 조정될 수 있습니다.
  • 세부 사항은 계약 시 협의를 통해 확정됩니다.
