유효기간 견적서 발행일로부터 30일

현황 분석

현재 네이버 캘린더에 등록된 시공/실측/상담 일정을 직원이 직접 확인한 뒤, 윙고(Wingo) 프로그램에서 수동으로 알림톡을 발송하는 방식으로 운영 중입니다. 일정이 많아질 경우 발송 누락 가능성이 있으며, 반복적인 수작업으로 인한 업무 비효율이 발생하고 있습니다.

자동화 목표

네이버 캘린더 일정 등록/변경 시 자동으로 알림톡 발송 트리거가 작동하여, 지정된 시간에 고객에게 알림톡이 자동 전송되는 시스템을 구축합니다.

프로젝트 개요

자동 발송 알림톡 (5종)
  • 시공 확정 알림톡: 시공 일정 확정 시 고객에게 날짜/시간/주소 안내
  • 설치 확정 알림톡: 설치 일정 확정 시 방문 예정 안내
  • 실측 일정 확정 알림톡: 실측 방문 일정 확정 시 안내
  • 계약 완료 알림톡: 계약 체결 완료 시 확인 안내
  • 상담 보류 안내 알림톡: 상담 보류 상태 안내 및 재상담 유도
처리 프로세스
  • 1. 일정 등록: 네이버 캘린더에 일정 등록 (일정 제목에 유형 구분 키워드 포함)
  • 2. 자동 감지: 서버에서 캘린더 일정 변경을 실시간 감지
  • 3. 트리거 실행: 일정 유형에 따라 해당 알림톡 템플릿 자동 매칭
  • 4. 알림톡 발송: 지정 시간에 카카오 비즈니스 채널을 통해 고객에게 자동 전송

제안 기술 스택

  • 캘린더 연동: 네이버 캘린더 CalDAV / API 연동
  • 알림톡 발송: 카카오 비즈니스 채널 알림톡 API (비즈메시지)
  • 서버: Node.js 또는 Python 기반 자동화 서버
  • 스케줄러: 일정 감지 폴링 + 발송 시간 예약 큐
  • 관리 화면: 발송 현황 조회 및 로그 확인 웹 대시보드
