프로젝트 개요

인플루언서 마케팅 플랫폼 "픽앤뷰"의 리뉴얼 프로젝트

기술 스택
  • 퍼블리싱: HTML5 + CSS3 + JavaScript + jQuery
  • 프론트엔드: JSP + jQuery
  • 백엔드: Spring Boot + MyBatis
  • 데이터베이스: MySQL
주요 기능
  • 사용자 웹: 홈, 리뷰, 캠페인, 유튜브, 마이페이지
  • 관리자 어드민: 전시관리, 회원관리, 콘텐츠관리, 게시판관리
  • SNS 콘텐츠 자동 수집 (인스타그램/유튜브)
  • SNS 소셜 로그인 (카카오/네이버)
  • 카카오 알림톡 연동
예상 기간

약 2개월 (45일 WBS 기준)
