주요 기능 분석
  • 메인 페이지: 회전식 배너, YouTube 채널 영상 임베드 (4개), 차량 검색 인터페이스
  • 차량 검색: 제조사 → 모델 → 세부모델 계층형 드롭다운, 가격대별 필터링 (9단계)
  • 차량 카테고리: 국산 / 수입 / 화물·특장 분류
  • 내차팔기: 판매 상담 신청 폼 (이름, 차량명, 연락처, 문의사항, 개인정보동의)
  • 최근 본 차량: 3개 슬롯 열람 이력 관리
  • 외부 연동: 카카오톡 상담, 네이버 블로그, 네이버 지도, YouTube 채널
  • 반응형: 모바일 (Android/iOS) 분기 처리
현재 기술 스택 (추정)
  • 프론트엔드: HTML/CSS + jQuery + 바닐라 JavaScript
  • 백엔드: PHP 또는 JSP 기반 MVC 구조 (추정)
  • 외부 API: YouTube IFrame API, 카카오톡 채널, 네이버 지도

프로젝트 개요

레퍼런스 사이트(중고차파괴자.com)와 동일한 기능을 갖춘 중고차 매매 플랫폼 신규 제작

제안 기술 스택
  • 프론트엔드: Next.js (React) + TypeScript + Tailwind CSS
  • 백엔드: Next.js API Routes
  • 데이터베이스: PostgreSQL (Supabase)
  • ORM: Prisma
  • 이미지 저장: Cloudflare R2 (30만장 이상 대용량 이미지 저장)
  • 외부 연동: YouTube IFrame API, 카카오톡 채널 API, 네이버 지도 API
  • 호스팅: Vercel (프론트) + Supabase (DB) + Cloudflare R2 (이미지)
기술 스택 선정 이유
  • SEO 최적화: Next.js SSR/SSG로 검색엔진 노출 극대화 (중고차 검색 유입 핵심)
  • 이미지 최적화: 차량 사진이 많은 사이트 특성상 next/image 자동 최적화 필수
  • 대용량 이미지: Cloudflare R2로 30만장 이상 이미지 저장 (이그레스 비용 무료)
  • 검색/필터링 UX: React 상태 관리로 실시간 필터링 구현 용이
  • 성능: 정적 생성 + 점진적 재생성(ISR)으로 빠른 페이지 로딩
  • 유지보수: TypeScript로 타입 안정성 확보, 버그 감소
  • 비용 최적화: Supabase + R2 이그레스 무료로 운영 비용 최소화
주요 구현 기능
  • 메인 페이지 (배너 슬라이더, YouTube 영상 섹션, 차량 검색)
  • 차량 검색 시스템 (제조사/모델/가격대 필터링)
  • 차량 목록 및 상세 페이지
  • 내차팔기 상담 신청 폼
  • 최근 본 차량 기능
  • 관리자 시스템 (차량 등록/수정/삭제, 상담 관리)
  • 반응형 웹 (PC/모바일)
