프로젝트명 P-BOX 일지봇 — 카카오 채널 OCR 자동화 시스템
프로젝트 유형 카카오톡 채널 챗봇 + GPT-OCR 기반 일지 사진 → 엑셀 자동 변환 시스템
유효기간 견적서 발행일로부터 30일

프로젝트 개요

현장에서 카카오톡 채널로 P-BOX 일지 사진을 전송하면, 서버에서 GPT-OCR로 표/텍스트를 자동 분석하여 엑셀 파일로 변환 후 카카오톡으로 결과를 회신하는 자동화 시스템을 구축합니다.

처리 프로세스
  • A. 사진 전송 (현장): 카톡 채널 "P-BOX 일지봇"으로 일지 사진 전송 → "접수 완료" 자동 응답
  • B. 자동 처리 (서버): 이미지 저장 → GPT-OCR 표/텍스트 추출 분석 → 텍스트/엑셀 파일 생성
  • C. 결과 회신 (카톡): 변환된 텍스트 표 전송 + 엑셀 파일(.xlsx) 링크 전송

제안 기술 스택

  • 챗봇 플랫폼: 카카오톡 채널 챗봇 API
  • OCR 엔진: GPT-4 Vision (이미지 → 표/텍스트 추출)
  • 서버: Node.js 또는 Python 기반 웹훅 서버
  • 엑셀 생성: openpyxl / ExcelJS 라이브러리
  • 파일 저장/전송: 클라우드 스토리지 (S3 또는 동등)
