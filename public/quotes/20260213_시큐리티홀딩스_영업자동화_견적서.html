프로젝트 개요

정보통신 유지보수 영업 프로세스를 자동화하여 영업 관리, 견적 발송, 계약서 생성, 알림, 리포트까지 일원화된 시스템을 구축합니다. Google Sheets를 관리대장으로 활용하고 Apps Script + 외부 API 연동을 통해 반복 업무를 제거합니다.

주요 기능
  • 영업 관리대장: 계약 현황, 상태(견적제출/협상중/계약완료), 만기 D-Day 자동계산, 조건부 서식
  • 견적 메일 자동 발송: 직원 체크 → 고객사명 자동 삽입된 표준 견적 메일 → 하이웍스 임시보관함 생성
  • 실시간 알림: 심층 상담 요청 시 대표에게 즉시 알림 (텔레그램/메일)
  • 사업자등록증 OCR: 이미지에서 사업자번호, 대표자명 자동 추출 → 시트 자동 입력
  • 계약서 PDF 자동 생성: 추출 정보로 신청서 템플릿 자동 채움 → PDF 변환 → 하이웍스 임시보관함 첨부
  • 일일 리포트: 매일 오후 6시, 당일 발송/처리 현황 요약 자동 전송

제안 기술 스택

  • 관리대장/데이터: Excel 또는 Google Sheets (조건부 서식, 수식 자동화)
  • 자동화 엔진: Google Apps Script
  • OCR: Google Vision API 또는 Naver Clova OCR
  • PDF 생성: Google Docs 템플릿 + Apps Script PDF 변환
  • 메일 연동: 하이웍스(Hiworks) SMTP / API
  • 알림: Telegram Bot API
  • 트리거: Apps Script Time-driven Trigger (일일 리포트)
