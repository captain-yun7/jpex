제1조 (목적)
본 계약은 "픽앤뷰" 리뉴얼 프로젝트 (용역계약서 PNV-2025-001) 수행 과정에서 갑이 을에게 제공하는 정보의 보호에 관한 사항을 정한다.
제2조 (기밀정보의 범위)
본 계약에서 "기밀정보"는 다음을 포함한다:
1. 소스코드, 데이터베이스 구조, 시스템 설계 자료
2. 회원정보 및 인플루언서 정보 (개인정보 포함)
3. 캠페인 정보 및 마케팅 자료
4. 서버 접속정보, API 키, 인증 정보
5. 갑이 기밀로 표시한 정보
제3조 (예외)
다음은 기밀정보에서 제외됩니다:
1. 이미 공개된 정보
2. 을이 적법하게 보유하고 있던 정보
3. 을이 독자적으로 개발한 정보
제4조 (비밀유지 의무)
1. 을은 기밀정보를 안전하게 관리하고 외부에 공개하지 않습니다.
2. 을은 기밀정보를 프로젝트 수행 목적으로만 사용한다.
3. 을은 프로젝트 참여 인력에게도 동일한 비밀유지 의무를 부여한다.
제5조 (개인정보 보호)
1. 을은 프로젝트 수행 중 취득한 개인정보를 안전하게 관리한다.
2. 을은 개인정보를 프로젝트 목적으로만 사용하며, 프로젝트 완료 후 즉시 삭제한다.
제6조 (자료 반환)
1. 을은 프로젝트 완료 시 또는 갑의 요청 시 제공받은 자료를 반환하거나 삭제한다.
2. 개인정보가 포함된 데이터는 즉시 완전히 삭제한다.
제7조 (보안 관리)
을은 다음 사항에 유의한다:
1. 소스코드를 공개 저장소에 업로드 금지
2. API 키, 비밀번호 등을 소스코드에 직접 작성 금지
3. 개인정보 테스트 후 즉시 삭제
4. 보안 문제 발생 시 즉시 통보
제8조 (손해배상)
을이 본 계약을 위반하여 갑에게 손해가 발생한 경우, 을은 이를 배상한다.
제9조 (유효기간)
본 계약은 계약일로부터 프로젝트 완료 후 2년간 유효한다.
제10조 (포트폴리오)
을은 본 프로젝트를 포트폴리오로 사용할 수 있으나, 개인정보 및 민감한 비즈니스 정보는 제외한다.
제11조 (기타)
본 계약과 관련된 분쟁은 상호 협의하여 해결하며, 협의가 이루어지지 않을 경우 관할 법원에서 해결한다.
※ 안내사항
• 본 계약은 용역계약서(PNV-2025-001)와 함께 적용됩니다.
• 프로젝트 참여 개발자는 별도 보안서약서를 작성한다.
• 기밀정보는 프로젝트 목적으로만 사용하며, 완료 후 삭제한다.

본 계약을 증명하기 위하여 계약서 2부를 작성하고, 갑과 을이 서명·날인 후 각 1부씩 보관한다.
