본인은 JPEX Studio의 일원으로서 픽앤뷰 리뉴얼 프로젝트 수행 중 알게 된 정보를 적절히 관리하고 보호할 것을 다음과 같이 서약합니다.
1. 기밀정보 보호
① 프로젝트 수행 중 알게 된 소스코드, 회원정보, 비즈니스 로직 등을 외부에 공개하거나 누설하지 않는다.
② 프로젝트와 관련된 정보를 개인적 용도로 사용하지 않는다.
③ 서버 계정, API 키 등 접속정보를 안전하게 관리한다.
2. 개인정보 보호
① 프로젝트 수행 중 접근한 회원 개인정보를 외부에 유출하지 않는다.
② 개인정보가 포함된 데이터를 개인 PC나 USB에 무단으로 복사하지 않는다.
③ 테스트 시 실제 회원정보 대신 더미 데이터를 사용하도록 노력한다.
3. 보안 준수사항
① 소스코드를 공개 저장소(GitHub Public 등)에 업로드하지 않는다.
② API 키나 데이터베이스 비밀번호를 소스코드에 직접 작성하지 않고 환경변수로 관리한다.
③ 보안 문제 발견 시 발주자에게 즉시 알리겠다.
4. 프로젝트 종료 시
① 프로젝트 완료 후 개인 PC에 남아있는 소스코드와 관련 자료를 삭제한다.
② 회원 개인정보가 포함된 데이터베이스 백업 파일이나 테스트 데이터를 삭제한다.
5. 포트폴리오 및 SNS
① 프로젝트 내용을 SNS나 블로그에 올릴 때는 회원정보 등 민감한 정보를 제외한다.
② 포트폴리오 작성 시 화면 캡처에 개인정보가 포함되지 않도록 주의한다.
6. 의무 지속
① 프로젝트 완료 후에도 알게 된 정보의 비밀을 유지한다.
② 특히 회원 개인정보는 영구적으로 보호한다.
※ 참고사항
기밀정보나 개인정보를 고의로 유출할 경우 관련 법령(부정경쟁방지법, 개인정보보호법 등)에 따라 법적 책임이 발생할 수 있으니 주의하시기 바랍니다.
