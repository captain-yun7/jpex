제1조 (계약의 목적)

본 계약은 갑이 을에게 상품권 판매/교환 플랫폼 개발 용역을 의뢰하고, 을이 이를 수행함에 있어 필요한 사항을 정함으로써 양 당사자의 권리와 의무를 명확히 하고 신의성실의 원칙에 따라 계약을 이행하는 것을 목적으로 한다.

제2조 (개발 범위 및 내용)

프로젝트명

도리콘 상품권 판매/교환 플랫폼

개발 범위
  • 상품권 판매 신청 시스템
    • 상품권 종류 선택 (이미지, 상품권명, 설명 표시)
    • 핀번호 직접 입력 (전체 붙여넣기 시 자리수별로 자동 분리)
    • 주문정보 입력 (은행명, 예금주명, 계좌번호, 휴대폰번호)
    • 개인정보 수집 및 이용 동의 체크박스 (동의서 전문 확인 가능)
    • 교환 유의사항 및 이용조건 동의 체크박스
  • 관리자 페이지
    • 상품권 종류 추가/수정 (이미지, 상품권명, 설명, 핀번호 자리수 등)
    • 교환 요청 데이터 확인 및 관리
    • 교환 완료 처리 기능
    • 실시간 판매내역 관리
    • 메뉴 및 문구 수정 기능
  • 실시간 채팅 시스템
    • 교환 신청 완료 시 채팅창 표시
    • 채팅창에 주문 이력 표시 (상품권 종류, 교환금액, 이용조건 동의여부, 상태)
    • 관리자 교환 알림 기능 (푸시 또는 이메일)
    • 교환 완료 시 채팅창에 자동 메시지 발송
  • 실시간 판매내역
    • 교환 완료 건 자동 표시 (최대 10-15건)
    • 오늘 날짜 자동 업데이트 표시
    • 판매 리스트는 날짜 미기재
  • 메뉴 및 페이지
    • 상품권 매입시세
    • 상품권 판매신청
    • 실시간 판매현황
    • 고객센터
    • 내 주문 조회하기

제6조 (완료 및 인수)

  • 을은 웹사이트를 완성하여 정상적으로 사용 가능한 상태로 제공한다.
  • 갑은 완성된 결과물을 확인하고, 수정이 필요한 경우 을에게 요청한다.
  • 양 당사자가 최종 완료를 확인한 시점에 인수가 완료된 것으로 본다.

제7조 (하자보수)

을은 인수 완료일로부터 3개월간 계약 범위 내 기능의 오류 및 버그를 무상으로 수정한다.

제외 사항: 새로운 기능 추가, 갑이 임의로 수정한 코드로 인한 오류

제8조 (지적재산권)

  • 개발된 소스코드의 저작권은 최종 대금 완납 시 갑에게 이전된다.
  • 소스코드는 갑이 필요시 요청하면 을이 즉시 제공한다.
  • 을은 본 프로젝트를 포트폴리오 용도로 사용할 수 있다.

제9조 (특약 사항)

  • 개인정보 수집 및 이용 동의서 전문은 갑이 별도로 제공한다.
  • 상품권 교환 유의사항 및 이용조건 문구는 갑이 별도로 제공한다.
  • 계약 범위를 벗어나는 추가 기능은 별도 협의한다.
  • 본 계약에 명시되지 않은 사항은 양 당사자 간 협의를 통해 해결한다.
계약 체결
