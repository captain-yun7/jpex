제1조 (계약의 목적)

본 계약은 갑이 을에게 정보통신 유지보수 영업 자동화 시스템 개발 용역을 의뢰하고, 을이 이를 수행함에 있어 양 당사자의 권리와 의무를 명확히 하는 것을 목적으로 한다.

제2조 (개발 범위)

정보통신 유지보수 영업 자동화 시스템

  • 영업 관리대장 — 계약 현황, 상태(견적제출/협상중/계약완료), 만기 D-Day 자동계산, 조건부 서식
  • 견적 메일 자동 발송 — 직원 체크 → 고객사명 자동 삽입된 표준 견적 메일 → 하이웍스 임시보관함 생성
  • 실시간 알림 — 심층 상담 요청 시 대표에게 즉시 알림 (텔레그램/메일)
  • 사업자등록증 OCR — 이미지에서 사업자번호, 대표자명 자동 추출 → 시트 입력
  • 계약서 PDF 자동 생성 — 추출 정보로 신청서 템플릿 자동 채움 → PDF 변환 → 하이웍스 임시보관함 첨부
  • 일일 리포트 — 매일 오후 6시, 당일 발송/처리 현황 요약을 대표에게 자동 전송

제3조 (계약금액)

총 계약금액 ₩1,500,000

결제 조건: 계약금 ₩500,000 / 잔금 ₩1,000,000

결제 방법: 계좌이체

※ 입금 계좌: 카카오뱅크 3333-35-5038363 제이펙스스튜디오(Jpex Studio)

제4조 (개발 일정)

총 개발 기간: 계약일로부터 4주 (1개월)

  • 1주차: 요구사항 확정, 관리대장 시트 설계, 템플릿 확정
  • 2주차: 관리대장 구축, 견적 메일 자동 발송, 알림 시스템 개발
  • 3주차: 사업자등록증 OCR, 계약서 PDF 자동 생성, 하이웍스 연동
  • 4주차: 일일 리포트, 통합 테스트, 사용 교육 및 최종 납품

제5조 (완료 및 하자보수)

  • 을은 시스템을 완성하여 정상적으로 사용 가능한 상태로 제공한다.
  • 을은 인수 완료일로부터 1개월간 오류 및 버그를 무상으로 수정한다.
  • 기능 추가 및 유지보수는 별도 협의 후 추가 비용이 발생할 수 있다.

제6조 (갑의 의무)

  • 갑은 하이웍스 계정 및 API 접근 권한을 제공한다.
  • 갑은 신청서/계약서 템플릿 원본을 제공한다.
  • 갑은 Excel 또는 Google Workspace 환경을 협의하여 제공한다.
  • 갑은 계약금액을 약정된 기한 내에 납부한다.

제7조 (기타)

  • OCR API 등 외부 API 사용료는 갑이 부담한다.
  • 계약 범위를 벗어나는 추가 기능은 별도 협의한다.
  • 을은 갑의 업무 관련 정보에 대해 비밀을 유지한다.
  • 본 계약에 명시되지 않은 사항은 상호 협의하여 해결한다.
